# Kit Pipeline

1. Load `youmind.api_key` from `~/.youmind/config.yaml`.
2. Validate Kit connection through YouMind.
3. Gather source material from YouMind/web if needed.
4. Write or adapt the article for Kit.
5. Convert Markdown to HTML.
6. Preview locally if useful.
7. Publish as public or private broadcast.
8. Return broadcast ID and public URL when available.
9. If the broadcast is private, point the user to `https://app.kit.com/campaigns`.
