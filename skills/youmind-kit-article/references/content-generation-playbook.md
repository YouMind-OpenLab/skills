# Kit Content Generation Playbook

1. Lock the angle in one sentence.
2. Write a subject line that feels natural in inbox + feed.
3. Open with a short note from the author, not a generic blog intro.
4. Break the body into compact sections with clear headings.
5. Produce preview text that complements, not repeats, the subject.
6. Confirm whether the user wants public or private mode before publishing if unclear.
7. If you publish in private mode, remind the user that Kit shows the draft in `https://app.kit.com/campaigns`.
