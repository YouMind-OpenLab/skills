# Beehiiv Pipeline

1. Load `youmind.api_key` from `~/.youmind/config.yaml`.
2. Validate Beehiiv connection through YouMind.
3. Gather source material from YouMind/web if needed.
4. Write or adapt the article for Beehiiv.
5. Convert Markdown to HTML.
6. Preview locally if useful.
7. Publish as draft or confirmed post.
8. Return post ID, status, and web URL when available.
