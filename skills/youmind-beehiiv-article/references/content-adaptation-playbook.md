# Beehiiv Content Adaptation Playbook

Use this when the source already exists.

1. Remove platform-specific cruft from the source article.
2. Rewrite the opening so it reads like a newsletter note, not a repost.
3. Compress overlong sections; Beehiiv readers scan fast.
4. Pull a subtitle and preview text that can survive inbox/feed display.
5. Convert to simple HTML and keep embeds conservative.
