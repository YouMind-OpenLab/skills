# YouMind Article Dispatch

One topic, every platform. Distribute your content to multiple platforms with AI-powered adaptation for each platform's audience.

## Supported Platforms

| Platform | Skill | Status |
|----------|-------|--------|
| Dev.to | youmind-devto-article | Available |
| Hashnode | youmind-hashnode-article | Available |
| WordPress | youmind-wordpress-article | Available |
| Ghost | youmind-ghost-article | Available |
| LinkedIn | youmind-linkedin-article | Available |
| X/Twitter | youmind-x-article | Available |
| Reddit | youmind-reddit-article | Available |
| Facebook | youmind-facebook-article | Available |
| Instagram | youmind-instagram-article | Available |
| WeChat | youmind-wechat-article | Available |

## How It Works

1. You provide a topic
2. Dispatch generates a content brief
3. Each platform skill independently adapts the content for its audience
4. Content is published (or drafted) on each platform
5. You get a summary of all results

## Usage

```
# Single platform
"Write about AI agents for Dev.to"

# Multiple platforms
"Publish about AI coding on Dev.to, LinkedIn, and X"

# All platforms
"Publish everywhere about the future of AI"
```

## Setup

Each platform skill needs its own configuration. Install the platform skills you want, then configure each one's `config.yaml` with your API credentials.
